# Titan Chess

A fully functional chess game built with Vanilla JavaScript, HTML and CSS.

## Features

- Player vs Player mode
- Player vs AI mode
- Three difficulty levels:
  - Easy (random moves)
  - Normal (prioritizes captures)
  - Hard (prioritizes king capture)
- Board orientation toggle
- Pawn promotion system
- Castling support
- Score tracking per difficulty
- Dynamic background toggle
- Fully responsive (mobile, tablet portrait & landscape)

---

## AI Logic

The AI uses three levels of decision making:

- **Easy:** Completely random move selection.
- **Normal:** Prioritizes capturing moves.
- **Hard:** Attempts to capture the king first, then prioritizes captures.

---

## Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
- Vanilla JavaScript (no libraries)

---

## Responsive Design

The layout adapts to:

- Desktop
- Tablet (portrait & landscape)
- Mobile

Flexbox is used to control board alignment and layout consistency.

---

## Controls

- Click a piece to select it.
- Click a valid square to move.
- Press `Esc` to deselect.
- Toggle board orientation with the side images.
- Select difficulty level before starting AI mode.
